# sport_einwahl
This project automates assigning schoolchildren to sports sections based on their preferences and group capacity limits. It ensures fairness, tracks satisfaction metrics, generates Excel/PDF reports with charts, and has practical application at my school while demonstrating optimization techniques in education.
